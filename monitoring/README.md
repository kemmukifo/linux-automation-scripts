# 🔍 JBoss Port Analyzer

Ferramenta avançada para análise de portas em ambientes JBoss/WildFly.

---

## 🎯 Objetivo

Identificar:

- Portas em uso
- Conflitos de configuração (offset duplicado)
- Ambientes mal configurados
- Processos utilizando portas

---

## 🚀 Uso

```bash
./jboss-port-analyzer.sh
