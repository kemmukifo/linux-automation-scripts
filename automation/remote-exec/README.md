# 🚀 Remote Script Executor

Script para execução remota de scripts em múltiplos servidores via SSH.

---

## 🎯 Objetivo

Automatizar execuções em massa em ambientes Linux, reduzindo esforço manual e erros operacionais.

---

## ⚙️ Como funciona

- Conecta via SSH em múltiplos hosts
- Executa um script local remotamente
- Retorna status individual por servidor

---

## 🚀 Uso

```bash
./remote-script-executor.sh <usuario> <script>
