# 🔄 RSYNC Command Generator

Ferramenta web simples para geração de comandos `rsync` de forma rápida, prática e sem erros.

---

## 🎯 Objetivo

Facilitar a criação de comandos `rsync` para cópia de múltiplas pastas entre servidores, evitando erros manuais e aumentando produtividade em operações de infraestrutura.

---

## 🖥️ Interface

A ferramenta permite:

- Definir usuário remoto
- Informar servidor de destino
- Escolher diretório de destino
- Adicionar múltiplas pastas de origem
- Gerar comando automaticamente pronto para uso
